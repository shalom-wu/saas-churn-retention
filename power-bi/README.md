# Power BI Package

[`saas_churn_retention.pbix`](saas_churn_retention.pbix) — a 3-page retention
dashboard over the full 7,043-customer table, built and saved in Power BI
Desktop. The PBIP text source sits alongside (generated by
[`../scripts/build_pbip.py`](../scripts/build_pbip.py)), so the semantic
model, DAX measures and page layouts are reviewable as code.

| File | Purpose |
|---|---|
| `saas_churn_retention.pbix` | The dashboard — open in Power BI Desktop (free) |
| `*.SemanticModel/`, `*.Report/`, `*.pbip` | Text-based source for the dashboard |
| [`dashboard_brief.md`](dashboard_brief.md) | Pages, KPIs, and the headline read |
| [`data_model.md`](data_model.md) | Tables, the one relationship, and the Python↔SQL handoff |
| [`dax_measures.md`](dax_measures.md) | Every measure, copy-paste ready |
| [`refresh_instructions.md`](refresh_instructions.md) | Regenerating inputs + refreshing |
| [`screenshots/`](screenshots) | Real Desktop captures of the three pages |

All inputs load from [`../data/powerbi/`](../data/powerbi) via a
`DataFolder` parameter — regenerate them any time with
`python scripts/run_sql.py`.
